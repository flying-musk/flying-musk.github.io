如何準備培訓機師考試

Go check the following projects:

  • Mental Calculation

    這個是長榮航空培訓機師考試模擬機關卡的心算測驗練習。（二位數乘法以及三位數加法）

    This is a practice of mental calculation for EVA Air Pilot Aptitude Evaluation(PAE).

  • Short-term Memory

    這個是長榮航空培訓機師考試模擬機關卡的短期記憶練習。（圖卡數字記憶）

    This is a practice of short-term memory for EVA Air Pilot Aptitude Evaluation(PAE).

  • Flight Management System

    這個是長榮航空培訓機師考試模擬機關卡的FMS輸入隨機字串練習。（打字）

    This is a practice of entering random strings into FMS for EVA Air Pilot Aptitude Evaluation(PAE).

  • Heading Calculation

    這個是立榮航空培訓機師考試ATR模擬機關卡的飛機航向心算測驗練習。

    This is a practice of heading calculation for UNI Air Pilot Aptitude Evaluation(PAE).

  • New Icon

    這個是中華航空培訓機師考試心理衡鑑的記憶力練習。（圖案的看過或沒看過辨識）

    This is a practice of memory (identify new or old icons) for China Airlines memory test.

  • Random Path

  • Licence Plate

  • Reverse It

    這個是中華航空培訓機師考試心理衡鑑的記憶力練習。（依隨機順序出現的方塊，必須記憶其出現順序，依照正序或反序點擊。）

    This is a practice of memory (memorize order of appearing cube) for China Airlines memory test.

  • Red Or Green

    這個是中華航空培訓機師考試心理衡鑑的記憶力練習。（隨機線條、字串、數字、車牌、算式的看過或沒看過辨識）

    This is a practice of memory (identify new or old random generated items) for China Airlines memory test.
